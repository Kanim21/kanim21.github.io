Multi-agent LangGraph system simulating hospital bed tracking, discharge planning, and transport dispatch behind a Model Context Protocol (MCP) tool layer (FastMCP). Static human-in-the-loop approval gates elevated-risk actions; every FHIR bundle is validated against R4B models, and free text passes through a Presidio-backed PHI guardrail before it ever reaches a log.

enterprise-network-platform

Multi-site enterprise network simulation for ~5,000 users: 2-spine / 4-leaf EVPN-VXLAN fabric with eBGP underlay, OSPF WAN, and IKEv2/BGP hybrid connectivity to AWS Transit Gateway. Automated with Nornir and Ansible; Batfish offline config analysis runs in CI to catch routing policy errors before they touch a device.
